##学习git和github常用操作

###关联远程仓库
```
git remote add origin https://github.com/swellwu222/git_learn.git
```
###克隆远程仓库
```
git clone https://github.com/swellwu222/git_learn.git
```
###提交文件到本地参考
```
git add .
git commit -m "修改记录"
```

###提交到远程master分支
```
git push -u origin master
```

###查看状态和改动
```
git status
git diff
```

###查看提交历史
```
git log
```
